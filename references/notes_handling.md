# 创作笔记差异化处理（v3.1/v3.3 硬规则）

## 分类

| 章节位置 | 处理方式 |
|----------|----------|
| 非本章最后一节 | 创作笔记在三个产物中**全部保留** |
| 本章最后一节 | 公众号代码版用 HTML 注释包裹；其他两版保留灰色块 |

## 非末节 · 三版全保留

### 手机友好版

文末 markdown 灰色引用块，标注"不用于公众号发布"。

### 公众号版 HTML

`<table bgcolor>` 灰色块（模板 1 灰色变体），竖线色 `#999`，背景色 `#F0F0F0`。

### 公众号代码版

`<table bgcolor>` 灰色块（模板 1 灰色变体），同上。

## 末节 · HTML 注释包裹

### 公众号代码版

创作笔记用 HTML 注释包裹：

```html
<!-- ⚠️ 以下内容为创作笔记，不用于公众号发布
...笔记内容...
-->
```

### 手机友好版

保留为文末灰色引用块。

### 公众号版 HTML

保留为 `<table bgcolor>` 灰色块。

## 严禁

- 末节公众号代码版未用 HTML 注释包裹（笔记会泄露给读者）
- 非末节三版中任一版丢失笔记（破坏作者审稿连续性）
- 在非末节采用"末节特殊策略"（仅适用于本章最后一节）

## 自检

交付前确认：

- [ ] 当前章节位置（末节 / 非末节）
- [ ] 三版产物对应的处理方式
- [ ] 末节公众号代码版 grep 验证注释包裹成功
