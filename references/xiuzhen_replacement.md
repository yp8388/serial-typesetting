# 修真词替换表（v3.3 硬规则）

## 作用域

**只对配图 prompt 内部生效**——避免图片生成被 content_filter 拒绝。
**正文、对话、独白 100% 保留原文**——修真词是小说风格本身，强行替换会破坏作品。

| 范畴 | 是否替换 |
|------|---------|
| 配图 prompt（图片生成提示词） | ✅ 必须替换 |
| 正文 / 对话 / 心理描写 | ❌ 原文保留 |
| 题词 / 目录 / 创作笔记 | ❌ 原文保留 |
| 节标题 / 副标题 | ❌ 原文保留 |

## 实战流程

1. **生成配图 prompt 前**先过一遍替换表
2. **正文 / 标题 / 目录**保留原文不动
3. **如果用户明确说"修真词全替换（正文也替换）"**——按用户意图，正文按替换表改写，并在创作笔记中标注"已修真替换"

## 子 session 失败时

- 子 session 生成修真图频繁被 content_filter 拒绝时，**主 Agent 接手单张重做**比重启子 session 更高效
- 重做时**prompt 必须先经替换表处理**再发
- 重做 2-3 次仍失败 → 改用风景 / 静物 / 动物特写等低敏感度题材，配图位置保留但图注改为"意境图"

## 已知误区

| 误区 | 正确做法 |
|------|----------|
| "修真替换表也作用于正文" | ❌ 错。**只用于配图 prompt** |
| "修真词全部删除" | ❌ 错。删除会破坏作品风格 |
| "修真图应该改用 3D 卡通避免过滤" | ❌ 错。质感风格与凡途 1960 不符 |
| "修真词只在 prompt 出现一次就没问题" | ❌ 错。任何一次出现都触发过滤 |
| "修真场景改用古代/武侠题材替换" | ❌ 错。修真替换表已覆盖，避免新增二次污染 |
