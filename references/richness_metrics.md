# 丰富度硬指标（v3.1/v3.3）

每节必须满足以下密度要求：

| 装饰元素 | 密度要求 | 模板 |
|----------|---------|------|
| 对话块 | 每节 ≥ 1 个 | 模板 1 |
| 关键短语高亮 | 每节 ≥ 2 个 | 模板 6 |
| 卷首引线 | 每 3 节 ≥ 1 个 | **模板 9（独立模板）** |
| 场景 dinkus | 每节 ≥ 1 个 | 模板 8 |
| 数字徽章 | 每节 1 个（章节大标题用） | 模板 2 |
| 配图嵌入 | 配图总数 = 节数 + 1（封面） | 模板 5 |

## 严禁

- 注释块超过 8 行
- 题词区少于 5 行或多于 10 行
- 目录区只列"一/二/三/..."（必须完整列出每节标题）
- 卷首引线全部省略（必须每 3 节 1 个）
- 关键短语高亮每节少于 2 个

## 段落瘦身硬规则

- 单段 ≤ 100 字（理想 30-80 字）
- 动作 / 心理 / 对话 / 叙事"四段分离"
- 对话必须独立成段
- 长短段落交替排布
- 段间距 ≥ 1.3em（建议 `margin: 18-22px 0`）

## 丰富度自检流程

1. 写完每个节，统计对话块/高亮/dinkus/卷首引线数量。
2. 不达标 → 立即补充，再继续。
3. 不可"攒到最后一起补"——会破坏阅读节奏。
