# 配图 5 元素清单（v3.3 强制）

每张配图生成前，必须先填写本清单，对照文字描写段逐项验证。

| 元素 | 必填内容 | 验证来源 |
|------|----------|----------|
| **人物** | 主角年龄/性别/穿着/表情 | 文字描写段 |
| **动作** | 具体姿势/手势/视线 | 文字描写段 |
| **场景** | 室内/室外/具体地点 | 文字描写段 |
| **服装道具** | 时代特征/材质/颜色 | 文字描写段 |
| **时代标志** | 至少 2 个时代元素 | 配图优先选清单 |

5 元素清单不通过 → 重写 prompt 再生成，不凑合。

## Prompt 5 类关键词（每张图必含）

1. **质感关键词**：`Kodachrome 64 胶片质感` / `吴印咸摄影风格` / `1960s 中国乡村纪录片` / `老照片颗粒感` / `胶片噪点` / `暖色温 4500K`

3. **风格关键词**：`写实摄影` / `纪实风格` / `自然光` / `生活化瞬间`
4. **构图关键词**：`中景` / `特写人物动作` / `环境交代` / `自然取景`
5. **负面提示词**：`避免塑料感/油画风/水墨淡彩/外国人/现代服饰/不锈钢/玻璃/瓷砖/混凝土高层/霓虹灯/赛博朋克`

```
{时代} + {场景} + {人物动作} + {服装道具}，
{质感关键词} + {风格关键词} + {构图关键词}，
```
