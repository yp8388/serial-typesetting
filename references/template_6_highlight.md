# 模板 6：关键短语高亮

**替代旧写法**：`<strong style="...;border-radius:2px">`

## HTML 模板

```html
<strong style="color:#00897B;background-color:#E0F2F1;padding:2px 6px;font-weight:700">关键短语</strong>
```

## 关键点

- 去掉 `border-radius:2px`（保留率低）
- 保留 `color` / `background-color` / `padding` / `font-weight`（微信保留）

## 数量要求

- 每节 ≥ 2 个关键短语高亮
- 选用标准：转折点、情感高潮、主题揭示、悬念钩子
- 全章通常 5-10 个
