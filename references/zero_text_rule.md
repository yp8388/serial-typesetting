# 文字区零文字规则（v3.3）

黑板 / 报纸 / 牌匾 / 对联 / 信封等文字区域：

- **必须用黑色条带 + 模糊线条代替完整清晰中文**
- Prompt 必须强调："100% 零文字""绝对不可读出任何文字"
- 模型普遍会渲染"乱码"（如"食堂""武汉大"），需要 2-3 次重做才能完全去除

## 常见需要重做的场景

| 场景 | 重做原因 | 解决方案 |
|------|----------|----------|
| 食堂 | 餐桌附近出现"食堂"二字 | 改用模糊条带 |
| 武汉大学校门 | 牌坊上出现"武汉大学" | 改用黑色条带覆盖 |
| 报纸头版 | 出现可识别标题 | 改用模糊线条 + 黑色条带 |
| 对联 | 出现完整对联文字 | 改用模糊竖线条 |
| 黑板 | 出现可识别的板书 | 改用粉笔痕迹抽象表达 |

## 1977 恢复高考题材特别提示

- 报纸头版"恢复高考"相关字样
- 准考证 / 录取通知书
- 教室黑板板书
- 教材封面标题

以上文字区全部按"零文字"处理。

## Prompt 必须包含的负面提示

```
100% 零文字，黑板/报纸/牌匾/对联/信封用黑色条带+模糊线条代替，
避免可识别的中文/英文/符号
```
