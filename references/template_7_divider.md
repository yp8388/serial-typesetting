# 模板 7：分隔线

**替代旧写法**：`<hr style="border-top:1px dashed #C8E6E0">`

## HTML 模板

```html
<p style="text-align:center;margin:40px 0;color:#00897B;letter-spacing:8px;font-size:14px">——  ◆  ——</p>
```

## 关键点

- 完全不用 `<hr>`，改用文本符号 + 主色 + 字间距 8px 模拟分隔线效果
- `letter-spacing:8px` 制造呼吸感
- 主体符号 `——  ◆  ——` 或 `—— ※ ——` 均可
