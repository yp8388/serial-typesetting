# 模板 9：卷首引线块（v3.1 特色元素 · v3.3.2 恢复）

## 是什么

每节（或每 3 节）开头的一个**灰底 + 左竖线 + 斜体引用**块，用于：
- 设置本节的氛围 / 基调
- 放一句与本节主题相关的"金句"或环境描写
- 类似于书籍中每章开头的"题记"或杂志文章的"导语"

**这是 v3.1 最有特色的视觉元素之一——不要省略。**

## 与其他元素的区别

| 元素 | 位置 | 用途 | 模板 |
|------|------|------|------|
| **卷首引线** | **每节开头（或每 3 节 1 次）** | 本节氛围/金句 | **模板 9（本文）** |
| 题词区 | 全章开头一次 | 全章主旨/格言 | 模板 3 |
| 对话块 | 对话出现处 | 角色对话 | 模板 1 |
| 场景描写块 | 场景段落 | 环境/心理 | 模板 1 变体 |

## HTML 模板

```html
<table cellspacing="0" cellpadding="0" border="0" width="100%" style="border-collapse:collapse;margin:24px 0 16px">
  <tr>
    <td width="4" bgcolor="#00897B" style="width:4px;font-size:0;line-height:0">&nbsp;</td>
    <td style="padding:16px 20px;background-color:#F5FBFA;color:#666;font-size:14px;line-height:1.85;font-style:italic">
      <p style="margin:0;text-indent:0;text-align:left">
        这里的风带着山里的湿气，吹得人骨头缝里都发凉。
      </p>
    </td>
  </tr>
</table>
```

## 关键点

- **背景色 `#F5FBFA`**（与对话块一致，不是 `#FAFCFC` 近白色）
- **左侧 4px 主色竖线** `#00897B`
- **斜体** `font-style:italic`（与正文区分）
- **字号 14px**（比正文 15px 小一级，营造"引语感"）
- **颜色 `#666`**（次要色，不抢正文焦点）
- **上边距 24px，下边距 16px**（上宽下窄，与后续正文自然衔接）

## 密度要求

- 每 3 节 ≥ 1 个（不必每节都有，避免视觉疲劳）
- 放置位置：**章节大标题（模板 2）之后、正文第一段之前**
- 内容来源：从本节正文中挑选最有氛围感的一句（环境描写 / 心理独白 / 旁白金句）
- 长度：1-3 行，不超过 80 字

## 变体：无特殊变体

卷首引线统一使用上述标准样式。不需要像对话块那样分"场景变体""笔记变体"。

## 常见错误

| 错误 | 正确做法 |
|------|----------|
| 用对话块（模板 1）代替 | 卷首引线是独立元素，有专属模板 9 |
| 放在节末 | 必须放在**节开头**（标题后、正文前） |
| 用正文色 `#3F3F3F` | 用次要色 `#666` + 斜体 |
| 背景色用 `#FAFCFC`（近白） | 用 `#F5FBFA`（与对话块一致的浅青绿） |
| 省略不写 | 丰富度硬指标要求每 3 节 ≥ 1 个 |
