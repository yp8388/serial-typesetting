# 凡途 1960 人物基线（v3.3 硬定）

## 陆尘（15-18 岁 · 1977 恢复高考那年正好高三/初考）

- **年龄**：15-18 岁
- **身高**：约 165-170cm
- **脸型**：清瘦、颧骨略高（湘西少年典型）
- **表情**：安静、专注、带一点忧郁；眼睛里偶尔有"光"（玉坠灵气的暗示）
- **春夏装**：白色衬衫 + 蓝灰长裤 + 解放鞋
- **秋冬装**：藏蓝色棉袄 / 军绿色旧棉袄
- **必带道具**：**背土布书包**（不是现代帆布包/军绿帆布包/双肩包）
- **修真心境**：颈间挂温润古玉（淡淡青绿光晕）

## 时代禁忌

- 禁止绿色卫衣 / 帆布书包 / 双肩包 / 现代白色建筑 / 汽车 / 水泥瓷砖
- 禁止黑板 / 报纸 / 牌匾 / 对联 / 信封上有**完整清晰中文**（必须用黑色条带 + 模糊线条代替）
- 配图描述必须包含："1977 + 陆尘 15-18 岁 + 时代特征 + 场景"——少一个都不行

## 大黑 = 纯黑色土狗

- **大黑是狗的名字**（因为是黑色）——不是形容词
- **设定**：**纯黑色中华田园犬 · 毛色黑得发亮 · 四肢粗短**
- 每张图涉及"大黑"时 prompt 必须包含：`"纯黑色中华田园犬·毛色黑得发亮·四肢粗短"`
- **教训**：早期 2 张图 prompt 写成"黄色土狗"——**必须查原文确认动物颜色**（别想当然）

## 5 节统一色板（严禁按节分色）

| 颜色角色 | 色值 | 出现位置 |
|----------|------|----------|
| 主棕 | `#8B6F47` | 肤色 / 土墙 / 木质结构 |
| 土黄 | `#C9A961` | 草帽 / 麻袋 / 粗布 / 光斑 |
| 暖灰 | `#6B5D4F` | 阴影 / 石板 / 远山轮廓 |
| 墨绿 | `#5B6E4A` | 山林 / 田地 / 苔藓 |
| 银白 | `#D8D2C4` | 棉布衣物 / 晨雾 / 月光 |

- **色温**：4500K（暖白光），不要冷光不要霓虹
- **饱和度**：中等偏低（模拟褪色老照片），不要鲜艳饱和
- **季节暗示**：春=嫩芽+薄衣/夏=草帽+短袖/秋=落叶+夹袄/冬=棉袄+火塘
